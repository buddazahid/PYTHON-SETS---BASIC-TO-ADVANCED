# Python Sets — Basic to Advanced

This repository contains a single example file demonstrating Python set usage, from basic operations to more advanced patterns and idioms.

Contents
- `PYTHON SETS - BASIC TO ADVANCED.py` — a self-contained script with examples, explanations, and small snippets you can run.

Prerequisites
- Python 3.7 or later.

Run the examples
1. Open a terminal in this folder.
2. Run:

```
python "PYTHON SETS - BASIC TO ADVANCED.py"
```
